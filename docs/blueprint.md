# **App Name**: OccasionCraft

## Core Features:

- Event Creation: Vendors can create events with details like name, description, date, time, and location, using a UI/UX similar to Luma.com/create.
- Ticket Design: Vendors can design custom tickets with a user-friendly interface, incorporating various templates and design elements.
- Event Listing & Discovery: Users can browse and search for events based on category, location, date, and keywords, with a structure similar to Eventbrite.
- User Dashboard: Registered users can manage their purchased tickets, view event details, and receive event updates.
- Vendor Dashboard: Event vendors can manage their events, track ticket sales, add event ushers, and view analytics. Includes event ticket scanner functionality, automatic PDF ticket generation with designed theme and QR code, and a "Scan event or validate event ticket" feature.
- Admin Backend: Admin users can manage vendors, events, user accounts, and website settings through a secure and intuitive admin panel.
- Payment Gateway Integration: Integrate Paystack payment gateway for secure ticket purchases and vendor payouts.

## Style Guidelines:

- Primary color: Medium blue (#64B5F6), inspired by the logo, for a trustworthy and inviting feel.
- Background color: Very light blue (#F0F8FF), a desaturated tint of the primary color, for a clean backdrop.
- Accent color: Light sky blue (#81D4FA), analogous to the primary with lighter brightness, for highlights and CTAs.
- Headline Font: 'Poppins' (sans-serif) for headlines and titles.
- Body Font: 'PT Sans' (sans-serif) for body text.
- Use a consistent set of modern icons, and incorporate Eventbrite-style icons.
- Employ a grid-based layout similar to Eventbrite for event listings and detail pages.
- Adopt a clean and intuitive layout similar to Luma.com for the event creation page.
- Use subtle transitions and animations for a smooth and engaging user experience.
